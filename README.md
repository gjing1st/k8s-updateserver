#
## 🌱1基本介绍
### 1.1 项目目的
业务部署在kubesphere中，为方便业务系统升级和业务系统部署。使用go语言开发此升级服务和saas化接口
### 1.2 项目简介
基于以上目的，升级服务和k8s部署工具单独打包

## 2 使用说明
- 升级服务可通过Dockerfile打包成镜像，随业务系统一起部署至k8s中
- 部署工具至cmd/kubetool/kubetool.go打包
